RAVE (REST API Very Epic)

All paths are relative to https://api.foo.bar/v1
An API key is required, to get one contact mail@foo.bar
For JSONP response include &callback=<custom name>

GET /foo/bar/:id

Get foo bar based on id

Method parameter Type Required Description
id int Yes Bar id
Query parameter Type Required Description
i ing Yes Some required integer
s string No Some string to use for something

Example

https://foo.bar/v1/foo/bar/1234?i=665&s=Neighbour of the beast&key=1234
